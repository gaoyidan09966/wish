# Wish - 愿望墙项目

这是一个基于 **Vue** 和 **Node.js** 开发的愿望墙全栈项目。


## 演示视频
- [点击查看 Bilibili 项目演示](https://space.bilibili.com/1772424360)

## 目录结构
* `wish-admin-web`: 许愿墙管理后台登录页面
* `wish-admin-api`: 后端 API 接口
* `wish`: 许愿墙
